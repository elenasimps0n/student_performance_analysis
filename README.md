# Student Performance Analysis

An analysis of 6,607 students exploring the relationship between physical activity and exam scores.

## Dataset
Student Performance Factors: sourced from Kaggle.
https://www.kaggle.com/datasets/grandmaster07/student-exam-performance-dataset-analysis?resource=download&select=StudentPerformanceFactors.csv

## Findings
Students who engage in more physical activity tend to achieve higher average exam scores, with a notable positive trend from 2-6 hours per week.
